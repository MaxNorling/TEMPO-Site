Grunder

I dagens avsnitt börjar vi med vårt projekt. Vi skapar tre väsentliga klasser för vårt spel, Window, GameEngine och GamePanel. Window-klassen kommer skapa ett fönster på skärmen. GamePanel kommer förvara all vår framtida bildinformation och positioner till alla våra sprites. GameEngine kommer vara vårt tick-system. Ett tick-system gör så att spelet uppdateras ett antal gånger per sekund, i vårt fall kommer det vara 60-gånger per sekund.

package tempo;

import javax.swing.JFrame;

/**
 *
 * @author Oscar Almqvist
 */
public class Window extends JFrame {
    
    private GamePanel gp;
    private GameEngine ge;
    
    public Window() {
        gp = new GamePanel();
        ge = new GameEngine(gp);
        setTitle(Tempo.NAME);
        setSize(Tempo.WIDTH, Tempo.HEIGHT);
        setVisible(true);
        setResizable(false);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        add(gp);
        
    }

}
